Zinc - H2O System

Description of method

- Mass Balance:

[ZnII]Total = [Zn2+] + [ZnOH+] + [Zn(OH)2] + [Zn(OH)42-]

- Chemical Reactions:

1. ZnO + 2H+ = Zn2+ + H2O

2. Zn(OH)2 + 2H+ = Zn2+ + 2H2O

3. ZnO + H+ = ZnOH+

4. Zn(OH)2 + H+ = ZnOH+ + H2O

5. ZnO + H2O = Zn(OH)2 (undiss.)

6. Zn(OH)2 = Zn(OH)2 (undiss.)

7. Zn(OH)2 + 2H2O = Zn(OH)42- + 2H+

8. ZnO + 3H2O = Zn(OH)42- + 2H+

9. Zn2+ + H2O = ZnOH+ + H+

10. ZnOH+ + H2O = Zn(OH)2 (undiss.) + H+

11. Zn(OH)2 (undiss.) + 2H2O = Zn(OH)42- + 2H+

- Electrochemical Reactions:

1. Zn2+ + 2e- = Zn

2. ZnO + 2H+ + 2e- = Zn + H2O

3. Zn(OH)2 + 2H+ + 2e- = Zn + 2H2O

4. Zn(OH)42- + 4H+ + 2e- = Zn + 4H2O

5. O2 + 4H+ + 4e- = 2H2O

6. 2H+ + 2e- = H2
