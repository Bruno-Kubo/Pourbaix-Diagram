Silver - H2O System

Description of method

- Mass Balance:

[AgI]Total = [Ag+] + [AgOH] + [Ag(OH)2-] + [AgO-]

- Chemical reactions:

1. Ag2O + 2H+ = 2Ag+ + H2O

2. Ag2O + H2O = 2AgOH

3. Ag2O + 3H2O = 2Ag(OH)2- + 2H+

4. Ag2O + H2O = 2AgO- + 2H+

5. AgO + 2H+ = Ag2+ + H2O

6. Ag2O3 + 2H+ = 2AgO+ + H2O
