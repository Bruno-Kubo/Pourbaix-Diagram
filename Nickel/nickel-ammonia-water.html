Nickel - Ammonia - H2O System

Description of method

- Mass Balances:

[NH3]Total = [NH3] + [NH4+] + 4[Ni(NH3)42+] + 6[Ni(NH3)62+]

[NiII]Total = [Ni2+] + [Ni(OH)2] + [Ni(NH3)42+] + [Ni(NH3)62+]

- Chemical reactions:

1. NH3(g) = NH3(aq)

2. NH4+ + H+ = NH3

3. Ni(OH)2(c) + 2H+ = Ni2+ + 2H2O

4. Ni(OH)2 + 4NH4+ = Ni(NH3)42+ + 2H2O + 2H+

5. Ni(OH)2 + 6NH4+ = Ni(NH3)62+ + 2H2O + 4H+

6. Ni(OH)2(c) + 4NH3 + 2H+ = Ni(NH3)42+ + 2H2O

7. Ni(OH)2(c) + 6NH3 = Ni(NH3)62+ + 2H2O

- Electrochemical reactions:

1. Ni2+ + 2e- = Ni

2. Ni(OH)2 + 2H+ + 2e- = Ni + 2H2O

3. Ni(NH3)42+ + 4H+ + 2e- = Ni + 4NH4+

4. Ni(NH3)62+ + 6H+ + 2e- = Ni + 6NH4+

5. Ni(NH3)42+ + 2e- = Ni + 4NH3

6. Ni(NH3)62+ + 2e- = Ni + 6NH3

7. Ni(OH)3 + 4NH4+ + e- = Ni(NH3)42+ + 3H2O + H+

8. Ni(OH)3 + 6NH4+ + e- = Ni(NH3)62+ + 3H2O + 3H+

9. Ni(OH)3 + 6NH4+ + 3H+ + e- = Ni(NH3)62+ + 3H2O

10. Ni(OH)3 + 3H+ + e- = Ni2+ + 3H2O

11. Ni(OH)3 + H+ + e- = Ni(OH)2+ + H2O

12. O2 + 4H+ + 4e- = 2H2O

13. 2H+ + 2e- = H2
