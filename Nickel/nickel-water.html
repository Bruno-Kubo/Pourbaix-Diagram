Nickel - H2O System

Description of method

- Mass Balance:

[NiII]Total = [Ni2+] + [Ni(OH)2]

- Chemical reactions:

1. Ni(OH)2 + 2H+ = Ni2+ + 2H2O

- Electrochemical reactions:

1. Ni2+ + 2e- = Ni

2. Ni(OH)2 + 2H+ + 2e- = Ni + 2H2O

3. Ni(OH)3 + H+ + e- = Ni(OH)2 + 2H2O

4. Ni(OH)3 + 3H+ + e- = Ni2+ + 3H2O

5. O2 + 4H+ + 4e- = 2H2O

6. 2H+ + 2e- = H2
