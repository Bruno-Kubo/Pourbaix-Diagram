Nickel - Citrate - H2O System

Description of method

- Mass Balances:

[Citrate]Total = [H3Cit] + [H2Cit-] + [HCit2-] + [Cit3-] + [NiCit-] + [NiHCit] + [NiH2Cit+]

[NiII]Total = [Ni2+] + [Ni(OH)2] + [NiCit-] + [NiHCit] + [NiH2Cit+]

- Chemical reactions:

1. H3Cit = H+ + H2Cit-

2. H2Cit- = H+ + HCit2-

3. HCit2- = H+ + Cit3-

4. Ni2+ + Cit3- = NiCit-

5. Ni2+ + HCit2- = NiHCit

6. Ni2+ + H2Cit- = NiH2Cit+

7. Ni(OH)2(c) + 2H+ = Ni2+ + 2H2O

8. NiCit- + 2H2O = Ni(OH)2 + 2H+ + Cit3-

9. Ni2+ + H3Cit = NiH2Cit+ + H+

10. NiH2Cit+ = NiHCit + H+

11. NiHCit = NiCit- + H+

- Electrochemical reactions:

1. Ni2+ + 2e- = Ni

2. NiH2Cit+ + H+ + 2e- = Ni + H3Cit

3. NiHCit + 2H+ + 2e- = Ni + H3Cit

4. NiHCit + H+ + 2e- = Ni + H2Cit-

5. NiCit- + 2H+ + 2e- = Ni + H2Cit-

6. NiCit- + H+ + 2e- = Ni + HCit2-

7. NiCit- + 2e- = Ni + Cit3-

8. Ni(OH)2 + 2H+ + 2e- = Ni + 2H2O

9. Ni(OH)3 + 3H+ + e- = Ni2+ + 3H2O

10. Ni(OH)3 + H+ + e- = Ni(OH)2+ + H2O

11. Ni(OH)3 + H3Cit + 2H+ + e- = NiH2Cit+ + 3H2O

12. Ni(OH)3 + H3Cit + H+ + e- = NiHCit + 3H2O

13. Ni(OH)3 + H2Cit- + 2H+ + e- = NiHCit + 3H2O

14. Ni(OH)3 + H2Cit- + H+ + e- = NiCit- + 3H2O

15. Ni(OH)3 + HCit2- + 2H+ + e- = NiCit- + 3H2O

16. Ni(OH)3 + Cit3- + 3H+ + e- = NiCit- + 3H2O

17. O2 + 4H+ + 4e- = 2H2O

18. 2H+ + 2e- = H2
