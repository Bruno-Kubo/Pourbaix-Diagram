Silver - H2O System

Description of method

- Mass Balance:

[AgI]Total = [Ag+] + [AgOH] + [Ag(OH)2-] + [AgO-]

- Chemical reactions:

1. Ag2O + 2H+ = 2Ag+ + H2O

2. Ag2O + H2O = 2AgOH

3. Ag2O + 3H2O = 2Ag(OH)2- + 2H+

4. Ag2O + H2O = 2AgO- + 2H+

5. AgO + 2H+ = Ag2+ + H2O

6. Ag2O3 + 2H+ = 2AgO+ + H2O

- Electrochemical reactions:

1. 2H+ + 2e- = H2

2. O2(g) + 4H+ + 4e- = 2H2O

3. Ag2+ + 1e- = Ag+

4. Ag+ + 1e- = Ag

5. AgO(c) + 2H+ + 1e- = Ag+ + H2O

6. Ag2O + 2H+ + 2e- = 2Ag + H2O

7. AgO(c) + H2O + 1e- = Ag(OH)2-

8. Ag(OH)2- + 2H+ + 1e- = Ag + 2H2O

9. 2AgO + 2H+ + 2e- = Ag2O(c) + H2O

10. Ag2O3(c) + 2H+ + 2e- = 2AgO(c) + H2O

11. Ag2O3(c) + 6H+ + 4e- = 2Ag+ + 3H2O

12. Ag2O3(c) + 6H+ + 2e- = 2Ag2+ + 3H2O

13. AgO+ + 2H+ + 2e- = Ag+ + H2O

14. AgO+ + 2e- = AgO-

15. AgO+ + 2H+ + 1e- = Ag2+ + H2O

16. AgO- + 2H+ + 1e- = Ag + H2O
